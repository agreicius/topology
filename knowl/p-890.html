Next given any \(x\in X\text{,}\) let \(\mathcal{C}\) be the set of all connected sets containing \(x\text{.}\) Note that \(\\mathcal{C}\) is nonempty since \(\{x\}\in \mathcal{C}\text{.}\) The set \(C_x=\bigcup_{C\in \mathcal{C}}C\) is easily seen to be a connected component containing \(x\text{.}\) Thus every element of \(x\) is contained in a connected component.
