Theorem 2.12.10. Direct sum equivalence.

Let \(\{A_\alpha\}_{\alpha\in I}\) be a family of subgroups of the abelian group \(G\text{.}\)
  1. We have \(G=\sum A_\alpha\) if and only if for all \(g\in G\) we can write \(g=\sum_{k=1}^ng_{\alpha_k}\) for some elements \(g_{\alpha_k}\in A_{\alpha_k}\text{.}\)
  2. We have \(G\cong \bigoplus A_\alpha\) if and only if for all \(g\in G\) we can write \(g=\sum_{k=1}^ng_{\alpha_k}\) for some elements \(g_{\alpha_k}\in A_{\alpha_k}\) in a unique way.

Proof.

According to (2.12.2) the map \(\phi\circ \oplus A_\alpha\rightarrow G\) coming from the inclusions \(j\colon A_\alpha\hookrightarrow G_\alpha\) has formula
\begin{align*} \phi((g_\alpha)) \amp = \sum_{\alpha\in I}j_\alpha(g_\alpha)\\ \amp = \sum_{\alpha\in I}g_\alpha \amp (j_\alpha=\id\vert_{A_\alpha})\text{.} \end{align*}
From this it is clear that \(\phi\) is surjective if and only if every element \(g\in G\) can be so written, and bijective if and only if this expression is unique. The result follows.
