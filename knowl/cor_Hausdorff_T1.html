Corollary 2.5.10. Hausdorff versus \(T_1\).

Let \(X\) be a topological space.
  1. If \(X\) is Hausdorff, then \(X\) is \(T_1\)
  2. It is not the case that if \(X\) is \(T_1\text{,}\) then \(X\) must be Hausdorff.
Using logical shorthand, we have
\begin{align} \text{ Hausdorff} \amp\implies T_1 \tag{2.5.1}\\ T_1\hspace{5pt}\amp\not\!\!\!\!\!\implies \text{Hausdorff} \tag{2.5.2} \end{align}
