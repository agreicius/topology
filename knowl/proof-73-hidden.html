1. Assume \(X\) is second countable and let \(\mathcal{B}=\{B_n\}_{n=1}^\infty\) be a countable basis.
    For each \(x\text{,}\) the set \(\mathcal{B}_x=\{B\in \mathcal{B}\colon x\in B\}\) is easily seen to be a countable local basis at \(x\text{.}\) This proves \(X\) is first countable.
    Let \(X=\bigcup_{i\in I}U_i\) be an open cover of \(X\text{.}\) For each \(B_n\in \mathcal{B}\) we have \(B_n\in U_{i_n}\) for some \(i_n\in I\text{.}\) Since the \(B_n\) cover \(X\text{,}\) we obtain the countable subcover \(X=\bigcup_{n=1}^\infty U_{i_n}\text{.}\) This proves \(X\) is Lindel√∂f.
    For each \(B_n\in \mathcal{B}\) pick \(x_n\in B_n\text{.}\) Since \(\mathcal{B}\) is a basis, it follows that \(A=\{x_n\}_{n=1}^\infty\) is dense: indeed, given any \(x\) and open set \(U\ni x\text{,}\) we have \(x\in B_n\subseteq U\) for some \(B_n\text{,}\) in which case \(x_n\in U\text{.}\)
