Theorem 2.3.4. Covering map properties.

Let \(p\colon E\rightarrow B\) be a covering map.
  1. For all \(b\in B\text{,}\) the preimage \(p^{-1}(\{b\})\) is a discrete subspace of \(E\text{.}\)
  2. For all \(e\in E\) there exists an open set \(U\) containing \(e\) such that \(p\vert_U\colon U\rightarrow E\) is a homeomorphism: i.e., \(p\) is a local homeomorphism.
  3. \(p\) is an open map.
  4. \(p\) is a quotient map.
  5. If \(E\) is connected and \(\abs{p^{-1}(\{b_0\})}=k\) for some \(b_0\in B\) and \(k\in \Z_+\text{,}\) then \(\abs{p^{-1}(\{b\})}=k\) for all \(b\in B\text{.}\) We call \(p\) a \(k\)-fold covering in this case.

Proof.

The proofs of these statements are left as exercises.
