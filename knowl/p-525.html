We will define \(\phi\) by constructing a function \(\tau\colon P(X)\rightarrow H\) that satisfies the following conditions:
\begin{align} [f]=[g] \amp \implies \tau([f])=\tau([g])\tag{1.15.1}\\ \tau(f*g)\amp =\tau(f)*\tau(g) \tag{1.15.2}\\ \tau(f)\amp =\phi_1([f]_U) \text{ for all } f\in P(U; x_0,x_0) \tag{1.15.3}\\ \tau(f)\amp =\phi_2([f]_V) \text{ for all } f\in P(V; x_0,x_0) \text{.}\tag{1.15.4} \end{align}
With such a \(\tau\) in hand, we let \(\phi\colon \pi_1(X,x_0)\rightarrow H\) be defined as \(\phi([f])=\tau(f)\text{.}\) It is clear from properties (1.15.1)–(1.15.2) that \(\phi\) is a homomorphism; and properties (1.15.3)–(1.15.4) guarantee that \(\phi\circ j_k=\phi_k\) for \(1\leq k\leq 2\text{.}\)
