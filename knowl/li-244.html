Item 2.
Let \(C\subseteq Y\text{.}\) We have
\begin{align*} C \text{ closed in } Y \amp \iff Y-C \text{ open in } Y \\ \amp\iff Y-C=U\cap Y \text{ for some open } U\subseteq X\\ \amp \iff C=(X-U)\cap Y \text{ for some open } U\subseteq X\\ \amp \iff C=C'\cap Y \text{ for some closed } C'\subseteq X\text{.} \end{align*}
