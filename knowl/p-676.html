Lastly, we look at when \(f\) is a quotient map. If \(f\) is a quotient map, then so is the composition \(g=f\circ p\) by statement (3) of TheoremÂ 2.11.4. Assume \(g\) is a quotient map. Since \(g\) is surjective and \(g=f\circ p\text{,}\) it follows that \(f\) must be surjective. Now suppose \(U=f^{-1}(V)\) is open in \(Y\text{.}\) Since \(p\) is continuous, \(p^{-1}(U)\) is open in \(X\text{.}\) As we saw above, \(p^{-1}(U)=g^{-1}(V)\text{,}\) which is a saturated open set of \(X\text{.}\) Since \(g\) is a quotient map, the set \(g(g^{-1}(V))\) is open. But again, because \(g\) is surejective, we have \(g(g^{-1}(V))=V\text{.}\)
