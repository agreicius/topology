Theorem 1.12.10. Direct sum equivalence.

Let \(\{A_\alpha\}_{\alpha\in I}\) be a family of subgroups of the abelian group \(G\text{.}\)
  1. We have \(G=\sum A_\alpha\) if and only if for all \(g\in G\) we can write \(g=\sum_{k=1}^ng_{\alpha_k}\) for some elements \(g_{\alpha_k}\in A_{\alpha_k}\text{.}\)
  2. We have \(G\cong \bigoplus A_\alpha\) if and only if for all \(g\in G\) we can write \(g=\sum_{k=1}^ng_{\alpha_k}\) for some elements \(g_{\alpha_k}\in A_{\alpha_k}\) in a unique way.
