Solution 1.3.9.1.

Since \(p\) is just the restriction of the covering map from ExampleÂ 1.3.3 to the open set \(\R_+\text{,}\) we easily see that it satisfies (1)-(4). However, there is no open neighborhood of \((1,0)\) that is evenly covered by \(p\text{.}\) Indeed, assume by contradiction that \(U\) was such an open neighborhood. Shrinking if necessary, we can assume that \(U=\{(\cos 2\pi s, \sin 2\pi s)\colon s\in (-\epsilon, \epsilon)\}\text{,}\) where \(\epsilon < 1/2\text{.}\) But then among the slices in the preimage of \(U\) we would have the set \([0,\epsilon)\text{,}\) and \(p\vert_{[0,\epsilon)}\) is not a homeomorphism onto \(U\text{.}\)
