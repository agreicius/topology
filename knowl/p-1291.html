Implication: \((\Rightarrow)\text{.}\) Assume \(X\) is completely regular. Recall that in particular this means \(X\) is \(T_1\text{.}\) Define \(I\) to be the set of all pairs \((x, U)\) where \(x\in X\) and \(U\) is an open neighborhood of \(x\text{.}\) For each \(i=(x,U)\text{,}\) let \(f_i\colon X\rightarrow [0,1]\) be a function satisfying \(f(x)=1\) and \(f(X-U)=\{0\}\text{.}\) (Such a function exists since \(X\) is completely regular.) Now, for all \(x\in X\) and for all open sets \(U\) containing \(x\text{,}\) corresponding to the index \(i=(x,U)\) we have the function \(f_i\) which satisfies \(f_i(x)=1> 0\) and \(f_i(X-U)=\{0\}\text{.}\) We conclude by TheoremÂ 2.22.2 that the map \(f(x)=(f_i(x))_{i\in I}\) is an embedding of \(X\) into \(\R^I\text{,}\) and thus that \(X\) is homeomorphic to a subspace of \(\R^I\text{.}\)
