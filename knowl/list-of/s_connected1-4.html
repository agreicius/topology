Example 1.12.3. Examples.

  1. If \(\abs{X}\geq 2\text{,}\) then \(X\) is not connected with respect to the discrete topology.
  2. If \(X\) is infinite, then \(X\) is connected with respect to the cofinite topology.
  3. Let \(L\) be the graph of the equation \(y=0\) and let \(C\) be the graph of the equation \(y=1/(1+x^2)\text{.}\) The subset \(X=L\cup C\subseteq \R^2\) is disconnected.
  4. \(Q\subseteq \R\) is not connected. In fact if \(A\subseteq \R\) is any set that is not an interval, then \(A\) is not connected. (Define an interval of \(\R\) to be a set \(I\) satisfying the following property: if \(a,b\in I\) and \(a < b\text{,}\) then \([a,b]\subseteq I\text{.}\) This definition includes singletons and the empty set as sort of degenerate intervals.)
