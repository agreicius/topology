Let \(G\) be an abelian group. The following statements are equivalent.
  1. \(G\) is free abelian.
  2. There is a set \(I\) and tuple \((g_\alpha)_{\alpha\in I}\in \prod_{\alpha\in I}G\) such that the map
    \begin{align*} \bigoplus_{\alpha\in I}\Z \amp \rightarrow G\\ (m_{\alpha}) \amp \mapsto \sum_{\alpha\in I} m_\alpha g_\alpha \end{align*}
    is an isomorphism.
  3. There is a set \(I\) and tuple \((g_\alpha)_{\alpha\in I}\in \prod_{\alpha\in I}G\) such that for all \(g\in G\) we can write \(g=\sum_{\alpha\in I}m_{\alpha}g_\alpha\text{,}\) where \(m_\alpha=0\) for all but finitely many \(\alpha\in I\text{,}\) in a unique way.
  4. There is a set \(I\) and tuple \((g_\alpha)_{\alpha\in I}\in \prod_{\alpha\in I}G\) such that for any abelian group \(A\) the map
    \begin{align*} \Hom(G,A) \amp \rightarrow \prod_{\alpha\in I} A \\ \phi \amp \mapsto (\phi(g_\alpha))_{\alpha\in I} \end{align*}
    is a bijection. In other words, there is a tuple \((g_\alpha)_{\alpha\in I}\) such that given any abelian group \(A\) and any elements \(h_\alpha\in A\text{,}\) there is a unique homomorphism \(\phi\colon G\rightarrow A\) satisfying \(\phi(g_\alpha)=h_\alpha\text{.}\)
