According to (1.12.2) the map \(\phi\circ \oplus A_\alpha\rightarrow G\) coming from the inclusions \(j\colon A_\alpha\hookrightarrow G_\alpha\) has formula
\begin{align*} \phi((g_\alpha)) \amp = \sum_{\alpha\in I}j_\alpha(g_\alpha)\\ \amp = \sum_{\alpha\in I}g_\alpha \amp (j_\alpha=\id\vert_{A_\alpha})\text{.} \end{align*}
From this it is clear that \(\phi\) is surjective if and only if every element \(g\in G\) can be so written, and bijective if and only if this expression is unique. The result follows.
