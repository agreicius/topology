Assume \(x_i\rightarrow x\) for some net \((x_i)_{i\in I}\) in \(A\text{.}\) Given any open set \(U\) containing \(x\text{,}\) there is an \(i_0\in I\) such that \(x_j\in U\) for all \(j\geq i_0\text{.}\) In particular, we have \(x_{i_0}\in U\text{.}\) Since \((x_i)_{i\in I}\) is a net in \(A\text{,}\) we conclude that \(x_{i_0}\in\cap A\cap U\text{.}\)
