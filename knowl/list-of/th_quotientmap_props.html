Theorem 1.11.4. Quotient map properties.

  1. Let \(f\colon X\rightarrow Y\) be a continuous surjective function. The following statements are equivalent.
    1. \(f\) is a quotient map.
    2. \(f(U)\) is open for all open saturated sets \(U\subseteq X\text{.}\)
    3. \(f(C)\) is closed for all closed saturated subsets \(C\subseteq X\text{.}\)
  2. Let \(f\) be continuous and surjective. If \(f\) is open or closed, then it is a quotient map.
  3. The composition of two quotient maps is a quotient map.
Statement (3) is straightforward. Statement (2) follows easily from (1) using equivalent statements (b) and (c), respectively.

Proof of (1).

We prove \((a)\iff (b)\) and \((b)\iff (c)\text{.}\)
\((a)\implies (b)\text{.}\) Assume \(f\) is a quotient map. Let \(U\subseteq X\) be open and saturated. Since \(f\) is a quotient map, to show \(V=f(U)\) is open, it is enough to show \(f^{-1}(V)=f^{-1}(f(U))\) is open. I claim \(f^{-1}(f(U))=U\text{,}\) whence the claim follows. In general for any function of sets we have \(f^{-1}(f(U))\supseteq U\text{.}\) (Check for yourself.) Since \(U\) is saturated, the reverse inclusion also holds. Indeed, if \(x\in f^{-1}(f(U))\text{,}\) then \(f(x)\in f(U)\) and thus \(f(x)=f(u)\) for some \(u\in U\text{;}\) since \(U\) is saturated, we have \(x\in U\text{.}\)
\((b)\implies (a)\text{.}\) Assume \(U=f^{-1}(V)\) be open for \(V\subseteq Y\text{.}\) We wish to show that \(V\) is open in \(Y\text{.}\) First observe that \(f^{-1}(V)\) is saturated: if \(x\in f^{-1}(V)\text{,}\) then \(f^{-1}(\{f(x)\})\subseteq f^{-1}(V)\text{.}\) Since \(f\) satisfies \((b)\) and \(U\) is an open saturated set, we have \(f(U)\) open. I claim \(f(U)=f(f^{-1}(V))=V\text{,}\) showing \(V\) is open. In general we always have \(f(f^{-1}(V))\subseteq V\) for any function of sets. For the other inclusion, given \(y\in V\text{,}\) there is an \(x\in X\) such that \(f(x)=y\text{;}\) then \(x\in f^{-1}(V)\) and \(f(x)=y\text{,}\) showing \(y\in f(f^{-1}(V))\text{.}\)
\((b)\implies (c)\text{.}\) Let \(C\) be a saturated closed map. First I claim that the open set \(U=X-C\) is saturated. To see why, suppose \(y=f(x)\) for some \(x\in U\text{.}\) Since \(x\notin C\) and \(C\) is saturated, we must have \(f^{-1}(\{y\})\cap C=\emptyset\text{.}\) Thus \(f^{-1}(\{y\})\subseteq U\text{,}\) as desired. Next, I claim \(f(U)=Y-f(C)\text{.}\) We have already seen that if \(y\in f(U)\text{,}\) then \(f^{-1}(\{y\})\cap C=\emptyset\text{,}\) and hence \(y\notin f(C)\text{.}\) This proves \(f(U)\subseteq Y-f(C)\text{.}\) For the other inclusion, given any \(y\in Y-f(C)\text{,}\) by surjectivity there is some \(x\in X\) such that \(f(x)=y\text{;}\) since \(y\notin f(C)\) we must have \(x\in U\text{;}\) thus \(y\in f(U)\text{.}\) This proves \(f(U)=X-f(C)\text{.}\) Since \(f\) satisfies (b), \(f(U)\) is open. We conclude that \(f(C)\) is closed, as desired.
\((c)\implies (b)\text{.}\) This argument is essentially identical to the one above.
