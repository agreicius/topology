Assume \(f\colon X\rightarrow S^2-\{P,Q\}\) where \(X\) is compact. Pick a homeomorphism \(h\colon S^2-\{P\}\rightarrow \R^2\) sending \(Q\) to \(\boldzero\text{,}\) and let \(g=h\circ f\text{.}\) Observe that if \(g\) is nullhomotopic via the homotopy \(H\colon X\times I\rightarrow \R^2\text{,}\) then \(f\) is nullhomotopic via the homotopy \(h^{-1}\circ H\text{.}\) Furthermore, using LemmaÂ 1.10.6, we see that \(P\) and \(Q\) lie in the same component of \(S^2-f(X)\) if and only if \(h(Q)=\boldzero\) lies in the unbounded component of \(\R^2-h(f(X))\text{.}\) Thus it suffices to show that if \(X\) is compact and \(g\colon X\rightarrow \R^2-\{\boldzero\}\text{,}\) then \(g\) is nullhomotopic if \(\boldzero\) is in the unbounded component of \(\R^2-f(X)\text{.}\) We now show that this is the case.
