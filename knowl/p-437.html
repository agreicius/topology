  1. Since \(\id_G\) satisfies \(\id_G\circ i_\alpha=i_\alpha\) for all \(\alpha\in I\text{,}\) we see by uniqueness that \(\phi=\id_G\text{.}\)
  2. Consider the projection map \(\pi_{\alpha_0}\text{.}\) It is easy to see, using the definition of \(i_{\alpha_0}\) that
    \begin{equation*} \pi_{\alpha_0}\circ i_{\alpha}=\begin{cases} 0 \amp \text{if } \alpha\ne \alpha_0 \\ i_{\alpha_0} \amp \text{if } \alpha=\alpha_0. \end{cases} \end{equation*}
    Using the definition of \(\phi_\alpha\text{,}\) we conclude that \(\phi\circ i_\alpha=\phi_\alpha\) for all \(\alpha\in I\text{.}\) It follows from uniqueness that \(\phi=\pi_\alpha\text{.}\)
