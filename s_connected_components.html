    Section 2.14 Connected components and locally connected spaces

    Definition 2.14.1. Connected components.

    Let \(X\) be a topological space. A subset \(A\subseteq X\) is maximally connected (resp., maximally path connected) if \(A\) is connected (resp. path connected), and if for all connected (resp. path connected) sets \(B\text{,}\) if \(A\subseteq B\text{,}\) then \(A=B\text{.}\) A connected component (resp., path-connected component) of \(X\) is a maximally connected (resp. path connected) subset of \(X\text{.}\)

    Theorem 2.14.2.

    Let \(X\) be a topological space.
    1. All connected components of \(X\) are closed. (This is not necessarily the case for path-connected components.)
    2. If \(A\) is connected (resp. path connected) and \(C\) is a connected component (resp. path connected component) of \(X\text{,}\) then either \(A\cap C=\emptyset\) or \(A\subseteq C\text{.}\)
    3. The set of all connected components (resp. path-connected components) of \(X\) forms a partition of \(X\text{.}\) In other words, every element of \(X\) lies in a component (connected or path connected), and any two distinct components are disjoint.
    4. Each connected component of \(X\) is a partition of path-connected components.

    Proof.

    1. Let \(C\) be a connected component. Since \(C\subseteq\overline{C}\) and \(\overline{C}\) is connected, we have \(C=\overline{C}\text{.}\) Thus \(C\) is closed.
    2. Suppose \(A\) is connected. If \(A\cap C\ne\emptyset\text{,}\) then \(A\cup C\) is connected by (2) of Theorem 2.12.5. Since \(C\) is maximally connected, we have \(A\cup C=C\text{.}\) Thus \(A\subseteq C\text{.}\) A similar argument applies to path connected components.
    3. Let \(C_1, C_2\) be connected components. By the previous result, since \(C_2\) is maximally connected, if \(C_1\cap C_2\ne \emptyset\text{,}\) then \(C_1\subseteq C_2\text{.}\) Since \(C_1\) is maximally connected, \(C_1=C_2\text{.}\) This proves that any two distinct connected components are disjoint.
      Next given any \(x\in X\text{,}\) let \(\mathcal{C}\) be the set of all connected sets containing \(x\text{.}\) Note that \(\\mathcal{C}\) is nonempty since \(\{x\}\in \mathcal{C}\text{.}\) The set \(C_x=\bigcup_{C\in \mathcal{C}}C\) is easily seen to be a connected component containing \(x\text{.}\) Thus every element of \(x\) is contained in a connected component.
      Again, the argument for path connected components is very similar.
    4. For each \(x\in X\) let \(C_x\) denote the (unique) connected component containing \(x\text{,}\) and let \(P_x\) denote the (unique) path connected component containing \(x\text{.}\) Let \(C\) be a connected component of \(X\text{.}\) For any \(x\in X\) we have \(P_x\cap C\ne\emptyset\text{;}\) since \(P_x\) is path connected, it is connected, and hence \(P_x\subseteq C\) by (1) above. It follows that \(C=\bigcup_{x\in C}P_x\) is a partition of \(C\) by path connected components.

    Definition 2.14.3. Locally connected space.

    A topological space \(X\) is locally connected (resp. locally path connected) at a point \(x\in X\) if for all open sets \(U\) containing \(x\text{,}\) there is an open connected (resp. open path connected set) \(V\) such that \(x\in V\) and \(V\subseteq U\text{.}\) The space \(X\) is locally connected (resp., locally path connected) if it is locally connected (resp., locally path connected) at all \(x\in X\text{.}\)

    Remark 2.14.4. Connected does not imply locally connected.

    Somewhat curiously, a space being connected does not necessarily imply that the space is locally connected. For example, the topologist's sine curve \(\overline{S}\) is connected, but not locally connected. For example, take the point \(P=(0,0)\in \overline{S}\) and the open set \(U=[0,1]\times (-1/2,1/2)\cap \overline{S}\text{.}\) Graphically, it is easy to see that \(U\) has a connected component partition consisting of \(\{0\}\times (-1/2,1/2)\) and infinitely many connected segments of the graph of \(f(x)=\sin (1/x)\) (one segment for each interval of the form \((1/(\pi/6+\pi n), 1/(-\pi/6+\pi n)\) that lies in \([0,1]\)). Any open connected set containing \((0,0)\) would have to lie in the connected component \(\{0\}\times (-1/2,1/2)\text{;}\) but this is impossible since \(\{0\}\times (-1/2, 1/2)\) does not contain any open sets!
