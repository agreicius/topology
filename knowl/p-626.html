By definition, the basis for the \(K\)-topology contains all finite open intervals. It follows from (2.2.3) that \(\mathcal{T}\subseteq \mathcal{T}''\text{.}\) Next, using an argument similar to the one in ExampleÂ 2.2.10, we show that the basis element \((0,1)-K\in \mathcal{T}''\) cannot be covered by open intervals \((a,b)\text{.}\) Indeed, consider the element \(0\in (0,1)-K\text{.}\) Any open interval \((a,b)\) containing \(0\) must contain an element of the form \(1/N\) for some positive integer \(N > 1\text{.}\) Since \(1/N\notin (0,1)-K\text{,}\) it follows that \((a,b)\not\subseteq (0,1)-K\text{.}\) We conclude that \((0,1)-K\) is not covered by open intervals \((a,b)\text{.}\)
