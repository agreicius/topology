Example 1.19.11. \(\R_\ell\) is regular.

Prove: \(\R_\ell\) is regular.
Solution.
Since \(\R_\ell\) has a finer topology than \(\R\text{,}\) it is \(T_1\text{.}\) We will use the equivalence (1) from TheoremÂ 1.19.8. Given any \(a\in \R\) and open set \(U\ni a\text{,}\) we can find \(b\) such that \(V=[a,b)\subseteq U\text{.}\) But \(V=\overline{V}\) is closed! (Prove this.) Thus we have found an open \(V\) containing \(a\) such that \(\overline{V}\subseteq U\text{.}\)
