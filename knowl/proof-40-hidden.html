Let \(\mathcal{A}\) be the disjoint union of the \(G_\alpha\text{,}\) let \(W\) be the set of all reduced words on \(\mathcal{A}\text{,}\) and let \(S_W\) be the set of all permutations of \(W\text{.}\) In other words, \(S_W\) is the set of all bijections \(\sigma\colon W\rightarrow W\text{,}\) a group under function composition.
Fix \(\alpha\in I\text{.}\) We now define a map \(i_\alpha G_\alpha\rightarrow S_W\) that associates to each element \(g\in G_\alpha\) a permutation \(\sigma_g\colon W\rightarrow W\in S_W\text{.}\) The permutation \(\sigma_g=i_\alpha(g)\) is defined as follows:
\begin{align*} \sigma_e \amp =\id_W \\ \sigma_g((g_{\alpha_1},g_{\alpha_2},\dots, g_{\alpha_n}))\amp =\begin{cases} (g) \amp \text{if } n=0 \\ (g,g_{\alpha_1},g_{\alpha_2},\dots, g_{\alpha_n}) \amp \text{if } n\ne 0 \text{ and } \alpha\ne \alpha_1 \\ (gg_{\alpha_1},g_{\alpha_2},\dots, g_{\alpha_n})\amp \text{if } n\ne 0 \text{ and } \alpha=\alpha_1 \text{ and } g_{\alpha_1}\ne g^{-1} \\ (g_{\alpha_2},\dots, g_{\alpha_n}) \amp \text{otherwise}. \end{cases} \amp (\text{for } g\ne e) \end{align*}
It is now straightforward, if somewhat tedious, to show that \(\sigma_{gh}=\sigma_g\circ \sigma_{h}\) for all \(g,h\in G_\alpha\text{,}\) whence it follows that \(\sigma_g\) is indeed a bijection with inverse \(\sigma_g^{-1}=\sigma_{g^{-1}\text{,}\) and finally that \(i_\alpha\colon G_\alpha\rightarrow S_W\) is a group homomorphism.
