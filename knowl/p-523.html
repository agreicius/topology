First we introduce some useful notation. For a path \(f\in P(X)\text{,}\) we write \([f]\) to denote the path homotopy equivalence class of \(f\) in \(X\text{;}\) if \(f(I)\subseteq W\) for \(W\in \{U, V, U\cap V\}\text{,}\) then we write \([f]_W\) to denote the path homotopy equivalance class of \(f\) in \(W\text{.}\) We can use this notation to describe our maps \(i_1,i_2,j_1,j_2\) as follows:
\begin{align*} i_1([f]_{U\cap V}) \amp = [f]_{U} \amp i_2([f]_{U\cap V}) \amp =[f]_{V} \amp (f\in P(U\cap V))\\ j_1([f]_U) \amp = [f] \amp \amp \amp (f\in P(U)) \\ j_2([f]_V)\amp =[f] \amp \amp \amp (f\in P(V)) \text{.} \end{align*}
