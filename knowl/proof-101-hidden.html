First observe that \(\mathcal{F}\) is nonempty. Indeed, given any \(t_0\in T\text{,}\) the net \((f_i\vert_{\{t_0\}})_{i\in I}\) is just a net in the compact space \(X_{t_0}\text{,}\) and hence has a limit point \(g\text{.}\) It follows that \((\{t_0\}, g)\in \mathcal{F}\text{.}\) (There is a slight subtlety here, which is that the space \(X_{t_0}\) is homeomorphic to the (somewhat funny looking) product space \(\prod_{t\in \{t_0\}}X_t\) via the map \(f\mapsto f(t_0)\text{.}\))
Next we show that any chain in \(\mathcal{F}\) has an upper bound in \(\mathcal{F}\text{.}\) To this end, given any chain \(\mathcal{C}\) in \(\mathcal{F}\text{,}\) we define \((\widetilde{S},\widetilde{g})\) as follows:
  • \(\widetilde{S}=\bigcup\limits_{(S,g)\in \mathcal{C}}S\text{;}\)
  • the function \(\widetilde{g}\colon \widetilde{S}\rightarrow \bigcup_{t\in \widetilde{S}}X_t\) is defined as \(\widetilde{g}(s)=g(s)\text{,}\) where \((S,g)\) is any element of \(\mathcal{F}\) for which \(s\in S\text{.}\)
First observe that \(\widetilde{g}\) is a well-defined function: if \(s\in S\cap S'\text{,}\) where \((S,g),(S',g')\in \mathcal{F}\text{,}\) then since \(\mathcal{C}\) is a chain either \((S,g)\leq (S',g')\) or \((S',g')\leq (S,g)\text{;}\) assuming without loss of generality that \((S,g)\leq (S',g')\) we have \(g'\vert_S=g\) by definition, and hence that \(g(s)=g'(s)\text{.}\) Next we must show that \(\widetilde{g}\text{,}\) understood as an element of \(\prod_{t\in \widetilde{S}}X_t\text{,}\) is a limit point of the net \((f_i\vert_{\widetilde{S}})_{i\in I}\text{.}\) Using the usual product topology basis, it suffices to show that given any finite set \(F\subseteq \widetilde{S}\) and open set of the form \(U=\prod_{t\in F}U_t\times \prod_{t\in \widetilde{S}-F}X_t\text{,}\) and given any \(i\in I\text{,}\) there is an \(i'\geq i\) such that \(f_i\vert_{\widetilde{S}}\in U\text{.}\) Since \(F\) is finite, we have \(F\subseteq \bigcup_{k=1}^nS_k\) for some sets \(S_k\) satisfying \((S_k, g_k)\in \mathcal{C}\text{.}\) Since \(\mathcal{C}\) is a chain, there is an index \(k_0\in \{k_1,k_2,\dots, k_n\}\) such that \(S_{k_0}\supseteq S_k\) for all \(1\leq k\leq n\text{.}\) Thus we may assume there is an element \((S, g)\in \mathcal{F}\) such that \(F\subseteq S\text{.}\) By definition \(g\) is a limit point of the net \((f_i\vert_S)_{i\in I}\text{.}\) Since \(\widetilde{g}\vert_S=g\text{,}\) it follows that \(g\in \prod_{t\in F}X_t\times \prod_{t\in S-F}X_t\text{.}\) Thus there is an \(i'\geq i\) such that \(f_{i'}\vert_S\in \prod_{t\in F}X_t\times \prod_{t\in S-F}X_t\text{.}\) It follows easily that \(f_{i'}\vert_{\widetilde{S}}\in \prod_{t\in F}X_t\times \prod_{t\in \widetilde{S}-F}X_t\text{.}\) This proves \(\widetilde{g}\) is a limit point of \((f_i\vert_\widetilde{S})_{i\in I}\text{,}\) and hence that \((\widetilde{S}, \widetilde{g})\in \mathcal{F}\text{.}\) It is clear from the definition of \((\widetilde{S},\widetilde{g})\text{,}\) that it is an upper bound of \(\mathcal{C}\text{.}\)
Having shown that \(\mathcal{F}\) satisfies the chain condition for Zorn's lemma, we conclude that it contains a maximal element \((S^*, g^*)\text{.}\)
