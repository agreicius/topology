Assume \(f\) is continous and invertible throughout. To be a homeomorphism we need \(g=f^{-1}\) to be continuous. Since \(g(y)=x\) if and only if \(f(x)=y\text{,}\) we see that \(g^{-1}(A)=f(A)\text{.}\) Thus \(g\) continuous if and only if \(g^{-1}(U)\) is open for all open \(U\subseteq X\text{,}\) if and only if \(f(U)\) is open for all open \(U\subseteq X\text{,}\) if and only if \(f\) is open. The same argument with “closed” replacing “open” shows \(g\) is continuous if and only if \(f\) is closed. The equivalence of (1)-(3) now follows.
