Theorem 1.15.5. Heine-Borel theorem.

Let \(X=\R^n\text{.}\) For any \(A\subseteq X\) the following conditions are equivalent.
  1. \(A\) is compact.
  2. \(A\) is closed and bounded with respect to \(d\) (equivalently, with respect to \(\rho\)).

Proof.

It is clear that the empty set is compact, closed, and bounded. Henceforth, we assume \(A\) is nonempty.
Assume \(A\) is compact. Since \(\R^n\) is Hausdorff, we know from TheoremÂ 1.14.7 that \(A\) is closed. We now show that \(A\) is bounded. Pick any \(\boldx\in A\text{.}\) We have \(A\subseteq \bigcup_{n\in \Z_+}B_n(\boldx)\text{.}\) Since \(A\) is compact, we have \(A\subseteq \bigcup_{n=1}^N B_n(\boldx)=B_{N}(\boldx)\) for some \(N\text{.}\) It follows that \(d(\boldy,\boldy')\leq 2N\) for all \(\boldy, \boldy'\in A\text{,}\) in which case \(\operatorname{diam}(A)\leq 2N< \infty\text{.}\)
